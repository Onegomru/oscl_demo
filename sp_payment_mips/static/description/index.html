Tested on Odoo Vanilla (Community and Enterprise) with Ubuntu OS

How to configure...?


In Paystack portal copy secret key and configure in Odoo.


Go to website menu.


Go to payment provider menu.


Find paystack provider and open the form view.


Add secrete key from paystack. And add journal in second tab.


Now, Go to website and shop, Go to final checkout payment page, Here you can see paystack payment option is available.
